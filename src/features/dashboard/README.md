# Dashboard Feature

User dashboard providing quick access to all platform features.

---

## Components

### DashBoard.jsx
**Main Dashboard Page**
- Grid of feature cards with navigation
- Each card displays:
  - Feature title
  - Description
  - Optional badge indicator
  - Click to navigate

### DashboardCard
**Reusable Card Component**
- Title with icon
- Description text
- Navigation path
- Badge for notifications/status

### DashBoard/ (subdirectory)
Additional dashboard components and utilities

---

## Features Accessible from Dashboard
- 📊 Career Assessments
- 👨‍🏫 Mentorship Sessions
- 💬 Community Feed
- 🎯 Career Recommendations
- 📚 Learning Resources
- 🏫 College Search
- 📅 My Bookings
- ⚙️ Profile Settings

---

## How It Works
1. User lands on dashboard after login
2. Feature cards displayed in responsive grid
3. Click any card to navigate to that feature
4. Badges show pending actions/notifications
5. Quick access to all platform capabilities

---

## Navigation
Dashboard serves as the central hub connecting all user-facing features of the platform.
