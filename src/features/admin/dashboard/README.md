# Admin Dashboard Feature

Central admin dashboard providing overview and quick navigation.

---

## Components

### dashboard/
Admin dashboard components container

---

## Features
- 📊 Overview metrics
- 🔗 Quick navigation
- 📈 Key statistics
- ⚡ Recent activity

---

## Dashboard Sections

### Metrics Overview
- Total users
- Active sessions
- Pending approvals
- Recent signups

### Quick Actions
- Create user
- Review mentors
- Send announcement
- View analytics

### Activity Feed
- Recent platform activity
- Important alerts
- System notifications

---

## Navigation Hub
Quick links to all admin modules:
- Analytics dashboard
- User management
- Announcements
- Settings
- Activity logs

---

## How It Works
1. Admin logs in
2. Dashboard displays key metrics
3. Quick actions for common tasks
4. Navigate to detailed modules
