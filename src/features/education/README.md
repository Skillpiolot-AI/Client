# Education Feature

Educational content including workshops and video resources.

---

## Components

### AvailableWorkshops.jsx
**Workshop Listing**
- Browse upcoming workshops
- Workshop details:
  - Title and description
  - Date and time
  - Instructor info
  - Registration button

### VideoForm.jsx
**Video Content Form**
- Add/edit video content
- Video URL input
- Category selection
- Description field

### VideoList.jsx
**Video Gallery**
- List of educational videos
- Video thumbnails
- Click to play
- Category filter

---

## Features
- 📅 Workshop discovery
- 🎬 Video tutorials
- 📚 Educational content
- 👨‍🏫 Instructor-led sessions

---

## How It Works
1. Browse available workshops
2. View workshop details
3. Register for sessions
4. Access video content library
5. Watch tutorials and guides

---

## API Endpoints Used
- `GET /workshops` - List workshops
- `GET /videos` - Video library
- `POST /workshops/register` - Register for workshop
