# Forms Feature (Admin)

Admin data management forms for platform content.

---

## Components

### CareerForm.jsx
**Career Data Management**
- Add/edit career information
- Career descriptions
- Required skills
- Salary ranges

### CollegesManagement.jsx
**College Directory**
- Add/edit colleges
- College details
- Program information
- Location data

### CompaniesManagement.jsx
**Company Directory**
- Manage company listings
- Company profiles
- Job opportunities

### InterestManagement.jsx
**User Interests**
- Define interest categories
- User preference options
- Career matching data

### JobTitles.jsx
**Job Title Management**
- Job title database
- Industry categories
- Role descriptions

### SkillsManagement.jsx
**Skills Database**
- Manage skills catalog
- Skill categories
- Proficiency levels

### StrengthManagement.jsx
**Strengths Database**
- Personal strengths catalog
- Assessment mapping

### Questions.jsx
**Assessment Questions**
- Manage quiz questions
- Question categories
- Answer options
- Scoring logic

### Recommendation.jsx
**Recommendation Engine**
- Configure recommendations
- Matching algorithms
- Weight settings

### AddWorkshop.jsx
**Workshop Management**
- Create workshops
- Schedule settings
- Instructor assignment

---

## Features
- 📝 Career data management
- 🏫 College directory
- 🏢 Company listings
- 💼 Job titles database
- 🎯 Skills catalog
- ❓ Assessment questions
- 📅 Workshop creation

---

## How It Works
1. Admin selects data category
2. View existing entries
3. Add new or edit existing
4. Save changes to database
5. Data available platform-wide

---

## API Endpoints Used
- `GET/POST /admin/careers` - Careers
- `GET/POST /admin/colleges` - Colleges
- `GET/POST /admin/companies` - Companies
- `GET/POST /admin/skills` - Skills
- `GET/POST /admin/questions` - Questions
- `GET/POST /admin/workshops` - Workshops
