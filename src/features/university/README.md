# University Feature

University administration and teacher dashboard portal.

---

## Components

### UniAdminPortal.jsx
**University Admin Portal**
- Complete university administration
- Student management
- Course management
- Faculty management
- Reports and analytics
- Settings configuration

### TeacherDashboard.jsx
**Teacher Dashboard**
- Class management
- Student progress tracking
- Assignment management
- Session scheduling
- Gradebook access

### AdminUniversityManagement.jsx
**System Admin University View**
- Manage all universities
- University approval/rejection
- University verification
- Admin controls

---

## User Roles

### University Admin
- Manage university data
- Add/remove faculty
- Monitor students
- Generate reports

### Teacher
- Manage classes
- Track student progress
- Schedule sessions
- Grade assignments

### System Admin
- Oversee all universities
- Approval workflows
- Platform-wide settings

## API Endpoints Used
- `GET /universities` - List universities
- `GET /universities/:id` - University details
- `PUT /universities/:id` - Update university
- `GET /teachers/dashboard` - Teacher data
- `GET /students` - Student list
